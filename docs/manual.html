LumiNative

Unlock the full potential of your external displays on Apple Silicon.

Core Features

Native Control

Control external brightness exactly like your built-in display using DDC/CI commands. Tap the slider for smooth animations.

Adaptive Presets

The "Add Preset" menu intelligently detects your environment (Clamshell, Laptop, or Desktop) and hides irrelevant sliders.

Clamshell Mode

Full support for closed-lid operation. The app automatically hides internal controls when appropriate.

Silent Presets

Create custom brightness presets without eye strain. Editing is silent; changes apply only when you click save.

Detailed Usage

Dashboards & Presets

LumiNative V2 introduces a powerful new Dashboard. Click the grid icon to access it.

  • Favorites Cycle: Add presets to your favorites by entering "Edit" mode in the dashboard and clicking the Star icon.
  • Quick Cycle: Click the Star icon in the main slider view to instantly cycle through your favorite brightness presets (e.g., Day -> Night -> Gaming).
  • Adaptive UI: The dashboard automatically hides "Internal" or "External" labels if that monitor is not connected (e.g. Clamshell mode).
  • Creating Presets: Click "New Preset". Drag sliders to set values. Note that "Live Preview" is disabled for comfort.

Synchronization

We use a robust "Double-Tap" sync engine to ensure reliability.

  • Immediate Sync: Reads hardware state instantly on app launch.
  • Robust Check: Re-scans 2 seconds later for slow monitors.
