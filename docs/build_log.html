MonitorControl V2 Build Log

A complete history of changes, features, and fixes implemented in MonitorControl V2.

V2.21.0: Final Release & Polish

  • Adaptive Dashboard: Dashboard preset buttons now intelligently hide "Int" or "Ext" percentages based on active monitors (e.g. Clamshell Mode).
  • Documentation: Included comprehensive PDF User Manual and Build Log.
  • Packaging: Automated DMG creation and delivery to Downloads folder.

V2.20.0: Silent Presets

  • Disabled Live Preview: Dragging sliders in "Add Preset" no longer changes hardware brightness immediately. This prevents eye strain during configuration.
  • Refined UX: Changes only apply when the preset is Saved and Activated.

V2.19.0: Adaptive Presets UI

  • Clamshell Awareness: The "Internal Brightness" slider is hidden from the Add Preset menu when the lid is closed.
  • cleaner Interface: Reduces visual clutter by showing only relevant controls.

V2.18.0: Focus Logic Fix

  • Reliable Targeting: Fixed a bug where keyboard shortcuts sometimes failed to target external monitors.
  • Safety Fallback: If cursor position cannot be determined, control defaults to ALL external monitors.

V2.17.0: Clamshell & Focus

  • Focus Follows Mouse: Brightness keys (F1/F2) now control the monitor under your mouse cursor.
  • Auto-Refresh: App automatically detects lid close/open and updates the monitor list instantly.

V2.16.0: Robust Sync

  • Double-Tap Engine: Implemented a multi-stage synchronization strategy (Immediate + Delayed) to handle slow-waking monitors.
  • Race Condition Fix: Eliminated startup sync bugs by ensuring display detection is complete before reading hardware state.

V2.15.0: Startup Synchronization

  • Cold Start Fix: App forces a hardware sync 0.5s after launch to ensure internal state matches reality.

V2.14.0: Live Editing

  • Edit Mode: Enabled editing of existing presets (Name, Icon, Values).
  • Zero-Kickstart: Fixed issues where presets wouldn't apply correctly on first try.

V2.13.0: UI Fixes

  • Scrollable Add Preset: Fixed layout issues on smaller screens.
  • Pinned Footer: Action buttons are now always visible.

V2.12.0: Preset Overhaul

  • Custom Values: Added explicit sliders for setting preset target values.
  • Curated Icons: Added high-quality SF Symbols for preset customization.

V2.0 - V2.11: Foundation

  • Liquid Glass UI: Complete translucent redesign.
  • Native Internal Control: Bridge to `DisplayServices` for built-in screen control.
  • Smooth Transitions: 120Hz ProMotion support and easing functions.
  • Performance: Optimized DDC/CI communication.
